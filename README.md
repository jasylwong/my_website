## Objective

## <a name="running_the_program">Running the program</a>

If you wish to run this on your local host and customise it, you can use the following steps.

### Setup
```
$ git clone https://github.com/jasylwong/my_website.git
$ cd my_website
$ npm install
$ npm start
```
